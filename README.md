- `index.html`
- `style.css`
- `script.js`
- `responsavel-facilite.png`
- `assets/responsavel-facilite.png`
- `robots.txt`
- `sitemap.xml`
- `CNAME`
- `README.md`

Depois clique em **Commit changes** e aguarde alguns minutos.
